# Практика по Java для РТУ МИРЭА

Это репозиторий содержит учебные практические задания по предмету Java (РТУ МИРЭА).

## Структура репозитория

- `ListTask1/` — задания по спискам, двусвязным спискам, циклическим спискам.
- `ListTask2/` — собственная реализация структуры связанного списка.
- `Task11/` — простая программа со студентом.
- `Task13/` — классы `Person`, `Address`, `Shirt`.
- `Task14/` — отдельное задание по классам и методам.
- `Task15/` — калькулятор.
- `Task18/` — исключения (`Exception1`, `Exception2`, `Exception3`, `ThrowsDemo*`).
- `Task19/` — кастомное исключение `InvalidDocumentException`.
- `Task20/` — задачи с наследованием, `Animal`, `MinMax`, `Tester`.
- `Task22/` — стек.
- `Task23/` — очереди (массивная и связная реализация, АДТ).
- `Task26/` — задания с `Node` и `Tester`.
- `Task27/` — `MyHashtab`.
- `Task28/` — `Hashmap`.
- `Task29/` — `CitiesAndRoads`.
- `Task6/`, `Task7/`, `Task8/`, `Task9/` — задания по перемещаемым объектам, сортировкам, алгоритмам.

## Как запускать

Для каждого задания откройте соответствующую папку и компилируйте/запускайте `Main.java` или `Tester.java` через командную строку или IDE.

Пример команд (Windows PowerShell):

```powershell
cd "d:\Work\Development\MIREA-Java-Practice\Task9"
javac Main.java
java Main
```

## Описание

Этот проект служит портфолио практических работ по Java для обучения в РТУ МИРЭА. Код написан студентом для отработки тем: структуры данных, алгоритмы, ООП, исключения, коллекции и т.д.
